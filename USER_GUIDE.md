# AI Data Copilot 使用手册

## 产品概述

AI Data Copilot 是 AI 驱动的数据 Agent 平台。不仅是大模型赋能的 AI+BI 全场景数据消费产品，更是促进企业增长的数据 AI Agent。通过核心的**问数**、**报告**、**解读**三大 Agent 能力，让人人都能拥有超级数据分析师，真正释放数据价值。

## 快速开始

### 1. 上传数据

1. 打开首页，点击「开始分析」或导航栏「上传」
2. 拖拽 Excel (.xlsx/.xls) 或 CSV 文件到上传区域
3. 系统自动解析数据，显示字段列表和统计摘要
4. 支持多 Sheet 文件（自动合并，添加 sheet_name 列区分）

### 2. 查看仪表盘

上传后进入「仪表盘」页面，自动展示：

- **数据卡片**：数据总量、字段数量、平均值、数据完整度
- **可视化图表**：分类占比饼图、对比柱状图、趋势折线图
- **AI 分析**：点击「开始 AI 分析」，DeepSeek AI 自动生成洞察、风险和优化建议

### 3. AI Agent 对话

导航栏点击「AI 对话」进入 Agent 交互界面。Agent 会自动识别你的意图：

| Agent 类型 | 触发示例 | 返回内容 |
|-----------|---------|---------|
| **问数 Agent** | "哪些产品销量最高？""price 平均值？""华东有多少条？" | 精确答案 + 图表建议 + 追问 |
| **报告 Agent** | "生成一份分析报告""帮我总结数据特征""分析一下整体情况" | 多段报告 + 关键指标 + 结论 |
| **解读 Agent** | "深度解读这份数据""有什么规律？""为什么 A 比 B 好？" | 故事化解读 + 异常检测 + 机会点 |

每次回复都会附带追问建议按钮，点击即可继续深入探索。

### 4. 生成报告

1. 导航栏点击「报告」
2. 查看数据集概览
3. 点击「打印报告」
4. 浏览器弹出打印对话框，选择「另存为 PDF」保存

报告包含：封面、数据集信息、数据明细表格（前 100 条）。

## 配置说明

### DeepSeek API Key

在 Vercel 项目设置 → Environment Variables 中添加：

```
DEEPSEEK_API_KEY=sk-xxx
```

或在本地 `.env.local` 文件中配置。

### Supabase 数据库

在 Supabase 项目 SQL Editor 执行建表脚本，并关闭 RLS（行级安全）。

## 常见问题

**Q: 上传后数据摘要或图表标签显示乱码？**
A: 刷新页面重新上传。如果持续出现，请检查文件编码是否为 UTF-8。

**Q: 点击"开始 AI 分析"没反应？**
A: 检查 DeepSeek API Key 是否正确配置。确认后刷新重试。

**Q: AI Agent 回复"请先上传数据"？**
A: Agent 需要数据才能工作。刷新页面后需重新上传数据文件。

**Q: PDF 报告保存后中文是乱码？**
A: 请使用浏览器「打印 → 另存为 PDF」功能，而非直接下载。浏览器原生渲染中文无乱码。

**Q: 多 Sheet Excel 数据不全？**
A: 系统自动合并所有 Sheet，通过 sheet_name 列区分数值来源。可切换查看。
